FreeScreenshareAPI

FreeScreenshareAPI is an open source completely free screenshare API that can easily be used in your websites via an iframe!

Cool Stuff About This:

  • Private codes that can be used via a search parameter
  • Servers are completely free
  • No API keys required!
  • Full Source Code available here.

Stream Choppiness

We use TurboWarp's to host this, but TurboWarp limits how many times you can send data to the servers, so the streams will appear choppy. To help this, we need someone to host a fork of TurboWarp's servers without the limits so we can have less choppiness.
