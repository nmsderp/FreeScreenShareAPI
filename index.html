FreeScreenshareAPI

FreeScreenshareAPI is an open source completly free screenshare api that can easily be used in your websites via an iframe!

Cool Stuff About This:

  • Private codes that can be used via a search parameter
  • Servers are completly free
  • No API keys required!
  • Full Source Code avialable here.

Stream Choppiness

We use TurboWarp's to host this, but TurboWarp limits how many times you can send data to the servers, so the streams will appear choppy, so to help this, we need someone to host a fork of TurboWarp's servers without the limits, so we can have less choppiness.
